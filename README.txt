Hello! This is the source code of Glucobot.

Glucobot is a Telegram Bot aimed at helping diabetic patients keeping records of their glucose levels and their meds intake hours.
It is simple, if you want to register your glucose after or before a meal, just click the appropriate buttons and type your the corresponding number.
Glucobot will write this info into its database. Whenever you want to acces your information, Glucobot will write a .xlsx file with all the info you gave it.
You can try it yourself! Just look for its user name: @Main_Glucobot

Glucobot has been developed as a personal project. It is built with the library Aiogram.
